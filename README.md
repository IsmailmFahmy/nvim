# My NeoVim Config

## Install
### Linux
```bash
gh repo clone "IsmailmFahmy/nvim" ~/.config/nvim
```
### Windows

```pwsh
gh repo clone "IsmailmFahmy/nvim" $env:localappdata\nvim\
```
